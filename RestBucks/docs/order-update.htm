Update an order

  • Uri: order/{id}
  • Method: POST
  • Preconditions: The status of the order must be unpaid. Once an order is paid it can't be updated.
  • Note: The only thing you can update so far is the location.
  • Example payload 
    <order xmlns="http://restbuckson.net">
  <location>takeAway</location>
</order>
